# SOS:
A hobbyOS written in Rust.

## Current Progress:

- A Linked List Heap Allocator
- Multicore Support
- FAT32 Filesytem Support
- An ATA PIO driver
- A Virtio-gpu driver for qemu
- Full Paging Support
- Full 64 bit support
